      🔵 Blue Team

      선발 명단 (Starting XI)

        교체 명단 (3/6)

          📊 전체 출전 요약 (1~4쿼터 통합)

          💡 총 1쿼터 이하 출전 선수는 🚨 로 경고 표시됩니다.

          🪄 공평하게 배치

          📖 Tactical Board 사용법

          🎯 기본 시작하기

          • 자체전 / 매치전 토글 (좌측 상단)
            · 자체전: 빨강 + 파랑 두 팀 운용 / 매치전: 우리 팀 1팀만 (전체 피치)
          • 1Q ~ 4Q 탭으로 쿼터별 라인업 관리. 5쿼터 필요시 + 5Q

          👥 선수 명단 편집

          • 양쪽 사이드바에서 선발 11명 + 교체 3~6명 이름 입력 (교체는 ➕/✕ 버튼으로 슬롯 추가·삭제)
          • 구장 위 선수 마커를 드래그해서 위치 자유 이동
          • 구장 위 선수를 클릭한 뒤(노란 테두리로 선택됨) 다른 선수를 클릭하면 두 선수의 위치가 서로 바뀜 (같은 선수 다시 클릭 시 선택 취소)
          • 포메이션 버튼으로 4-4-2 / 4-3-3 / 4-2-3-1 등 즉시 적용
          • 📋 일괄 입력: 카톡 등에서 복사한 명단을 한번에 붙여넣기 (1., 1), (1), 1번 등 다양한 접두사 자동 인식)

          🔄 선수 교체

          • 위치만 바꾸기: 구장 위 선수를 클릭 → 다른 선수 클릭 (명단은 그대로, 자리만 교환)
          • 같은 쿼터 내 두 슬롯의 🔄 버튼을 차례로 클릭
          • 선발 ↔ 교체 클릭 시 선택:
            · 완전 교체 (1:1 자리바꿈)
            · 중간 교체 (둘이 각 0.5쿼터씩 출전)
          • 🔁 쿼터 인원 교체: 한 쿼터의 라인업 + 포메이션을 다른 쿼터와 통째로 swap

          🪄 자동화 기능

          • 🪄 공평하게 배치 (2쿼터부터 활성): 이전 쿼터 누적 출전이 적은 필드 선수 우선으로 라인업 자동 추천
            · GK는 지정 골키퍼로 자동 고정 (출전 균형 예외)
          • ⬇️ 이전 쿼터 복사: 직전 쿼터의 라인업·포메이션·작전판을 그대로 복사

          📊 출전 통계

          • 사이드바 하단에 실시간 누적 출전 시간 표시
          • 📊 출전 요약 버튼으로 전체 통계 모달
            · 🥅 골키퍼 (출전 균형 예외) / ⚽ 필드 선수 (적게 뛴 순 정렬, ≤1쿼터는 🚨 경고)

          📸 캡처 / 공유

          • 📸 전체 캡처: 모든 쿼터 라인업을 한 PNG로 합성해서 다운로드 → 카톡 공유
          • 📤 공유코드 복사: 모든 데이터를 코드로 변환 → 다른 기기에서 📥 코드 불러오기로 복원
          • 모든 변경 사항은 브라우저에 자동 저장 (재방문 시 그대로)

          ✏️ 전술 작전판 (그리기)

          • 구장 아래 ✏️ 그리기 토글로 화살표/선 그리기 모드 활성화
          • 색상 선택, ↩️ 되돌리기, 🧹 모두 지우기
          • 쿼터별 독립 보관 + 캡처 시 함께 포함됨

          🎨 팀 커스텀

          • 사이드바 상단 팀 이름 클릭하여 편집 (예: "우리팀" → "동남FC")
          • 팀 색상 피커(동그라미)로 유니폼 색상 변경 → 마커·통계·UI 자동 매칭

          💡 활용 팁

          • PC ↔ 폰 동기화: 공유 코드(📤📥)로 데이터 이동
          • 모바일에서는 사이드바가 구장 아래로 스크롤해야 보입니다
          • 매치전 모드 ↔ 자체전 모드 전환 시 포메이션 자동 재배치 옵션 제공
          • 잘못 변경했을 때는 🗑️ 초기화 (단, 모든 데이터가 사라지니 주의)

          🔁 쿼터 인원 교체

          📋 명단 일괄 붙여넣기

          카카오톡 등에서 복사한 참석자 명단을 한 줄에 한 명씩 붙여넣어 주세요.
          "1.", "1)", "1번", "(1)", "-", "*" 같은 접두사는 자동으로 제거됩니다. (선발 11명 + 교체 3~6명 = 최대 17명)
